Language Cookie

My Grim Dawn Gear check your native language with HTTP headers to offers you a service the most relevant as possible. The application save your language configuration in SYSLang_LANG cookie. You can change manually the language which update that cookie. The lifetime of this cookie is set to one year.

Session Cookie

My Grim Dawn Gear offers you to protected builds with password to update them later. To make change on you build, you have to sign first with the password previously defined. To recognize you, you session is store on the server, but it require the cookie PHPSESSID. The lifetime is set to the session. That mean when you close your browser, the cookie is destroy.

Google Analitics

To know more about the use of the Application My Grim Dawn Gear, i decide to include API Google Analytics. The API create two cookies _ga and _gat. All Google Analytics features are disabled as default.

The aim is to know how many people are using the application and how often. My Grim Dawn Gear contains many data which requires hardware performance and Internet Traffic. I will probably adjust the host depending of these informations.

How to check cookies

Chrome Browser :

  • Push on key F12
  • Go to tab Application
  • In left panel, in section 'Storage', you will find 'Cookies'


Firefox Browser :

  • Push on key F12
  • Go to 'tool settings' represent by a gear.
  • Check the option 'storage'
